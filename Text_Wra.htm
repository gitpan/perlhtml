Text_Wrap

NAME

Text::Wrap -- wrap text into a paragraph

SYNOPSIS

use Text::Wrap; $Text::Wrap::columns = 20; # Default print wrap("\t","",Hello, world, it's a nice day, isn't it?");

DESCRIPTION

This module is a simple paragraph formatter that wraps text into a paragraph and indents each line. The single exported function, wrap(), takes three arguments. The first is included before the first output line, and the second argument is included before each subsequest output line. The third argument is the text to be wrapped.
